Especifica o nome fornecido à captura instantânea que é criada depois que as mudanças são aceitas.

Este campo pode ser configurado de acordo com qualquer sequência especificada pelo usuário, fazendo referência a variáveis de ambiente e parâmetros de compilação. É recomendável incluir pelo menos uma variável cujo valor seja exclusivo para cada compilação, por exemplo, BUILD_NUMBER, BUILD_ID ou BUILD_URL, para assegurar que compilações consecutivas criem capturas instantâneas com nomes diferentes. Se esse campo estiver vazio, será usado um nome padrão para a captura instantânea. Para tarefas que usam uma Definição de compilação, o nome da captura instantânea é padronizado como "${buildDefinitionId}_#${BUILD_NUMBER}". Para tarefas que usam um Fluxo ou uma Área de trabalho do repositório, o nome da captura instantânea é padronizado como "${JOB_NAME}_#${BUILD_NUMBER}".

Exemplo: para uma tarefa chamada "JUnit Integration Build", caso esse campo seja configurado como "${JOB_NAME}_#${BUILD_NUMBER}_Snapshot", o nome da captura instantânea criada pela primeira compilação seria "JUnit Integration Build_#1_Snapshot".
