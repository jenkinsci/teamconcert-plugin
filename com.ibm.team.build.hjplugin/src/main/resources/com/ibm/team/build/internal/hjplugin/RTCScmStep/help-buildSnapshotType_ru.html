Выберите этот пункт для загрузки моментальной копии SCM Jazz в рабочую область.
Для использования UUID моментальной копии из предыдущего вызова шага Team Concert в этой же компоновке укажите ${env.team_scm_snapshotUUID} в качестве значения типа компоновки
Для использования другого UUID моментальной копии, укажите его в параметре компоновки 'buildSnapshotUUID'.
