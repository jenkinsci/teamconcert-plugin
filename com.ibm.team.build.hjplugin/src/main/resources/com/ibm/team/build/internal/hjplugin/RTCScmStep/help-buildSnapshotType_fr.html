Cette option permet de charger une image instantanée Jazz SCM dans l'espace de travail.
Vous pouvez spécifier un identificateur unique universel d'image instantanée d'un appel précédent d'étape Team Concert dans la même génération avec ${env.team_scm_snapshotUUID} comme valeur de type de génération.
Sinon, spécifiez l'identificateur unique universel d'image instantanée via le paramètre de génération 'buildSnapshotUUID'.
