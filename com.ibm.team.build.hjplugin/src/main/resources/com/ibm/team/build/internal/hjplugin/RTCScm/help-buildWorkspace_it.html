Il nome dello spazio di lavoro del repository della build all'interno del server RTC (Rational Team Concert). È necessario definire solo uno spazio di lavoro con il nome. Lo spazio di lavoro del repository della build deve collaborare con gli stream su cui eseguire la build.

In caso di requisiti specifici riguardanti la modalità di caricamento dello spazio di lavoro del repository di build (ad esempio: creazione cartelle del componente, utilizzo delle regole di caricamento, eliminazione della sandbox prima del caricamento, ecc.) creare una definizione di build che rispecchi tali requisiti. Utilizzare l'opzione per generare da una definizione di build anziché da uno spazio di lavoro.
