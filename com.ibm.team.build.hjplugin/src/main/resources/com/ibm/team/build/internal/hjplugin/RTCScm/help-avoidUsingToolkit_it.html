Ove possibile è consigliabile evitare l'utilizzo del Toolkit della build quando si eseguono attività sul principale. Ciò perché si trova ancora nella fase sperimentale. È necessario un server RTC 5.0 che fornisce alcuni dei servizi utilizzati.

Il toolkit della build non verrà utilizzato quando si esegue il polling dell'RTC e si conclude la build RTC. Il toolkit è comunque richiesto. Viene utilizzato per altre attività di configurazione sul principale (ad esempio, la convalida della connessione all'RTC, la definizione della build o dello spazio di lavoro). Viene inoltre utilizzato per le attività di check out generalmente eseguite sui nodi secondari.
