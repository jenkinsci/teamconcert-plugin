Especifique el área de proyecto o el área de equipo RTC que se asociará a este trabajo. El área de proyecto o equipo especificada se utilizará para resolver la corriente configurada en el trabajo.

Para el área de proyecto, especifique el nombre.

Para el área del equipo, especifique el nombre de todas las áreas de equipo en la jerarquía, empezando con el nombre del área de proyecto, con los nombres separados por "/". Para una instancia, Banca JKE/Desarrollo/Interfaz de usuario identifica el área de equipo 'Interfaz de usuario' que está en el área de equipo 'Desarrollo' en el área de proyecto 'Banca JKE'.
