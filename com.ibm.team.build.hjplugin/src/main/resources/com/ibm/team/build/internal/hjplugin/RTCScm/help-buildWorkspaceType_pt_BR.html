Selecionar uma área de trabalho de compilação apenas aceitará e buscará os conjuntos de mudanças de seus destinos de fluxo.

Se você tiver requisitos específicos relacionados a como a área de trabalho do repositório de compilação deve ser carregada (exemplo: criar pastas de componentes, usar regras de carregamento, excluir ambientes de simulação antes do carregamento, etc.), use a opção para compilar a partir de uma definição de compilação.
