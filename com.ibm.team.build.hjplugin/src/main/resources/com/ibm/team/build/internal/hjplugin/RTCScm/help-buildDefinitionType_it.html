La selezione di una definizione di build consentirà una migliore integrazione con la build ed il controllo origine di RTC. La selezione di una definizione di build Hudson/Jenkins consentirà di richiedere le build da RTC o Hudson/Jenkins. Inoltre, i risultati della build verranno creati in entrambi e collegati tra loro.

Con l'opzione Jazz Source Control, è possibile specificare uno spazio di lavoro di build insieme ad ulteriori opzioni di accettazione e caricamento.
