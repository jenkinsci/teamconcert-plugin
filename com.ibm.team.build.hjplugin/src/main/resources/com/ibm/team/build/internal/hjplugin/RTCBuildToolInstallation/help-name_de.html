Für die RTC-SCM-Unterstützung muss auf ein Build Toolkit zugegriffen werden können, das sich auf dem Knoten befindet, auf dem Builds erstellt werden. Sie können mehrere Versionen des Build Toolkit konfigurieren (z. B ein Toolkit für RTC 4.0.2 und eines für RTC 3.0.1.5). Sie müssen jeder Build-Installation einen Namen geben, damit Sie beim Konfigurieren von globalen Einstellungen, Jobs und Slave-Knoten das gewünschte Toolkit auswählen können.
