Wenn Sie diese Option auswählen, können Sie eine Jazz-SCM-Momentaufnahme in den Arbeitsbereich laden.
Sie können eine Momentaufnahme-UUID von einem früheren Aufruf eines Team Concert-Schritts im selben Build mit ${env.team_scm_snapshotUUID} als Wert für den Buildtyp angeben.
Geben Sie ansonsten die Momentaufnahme-UUID über den Buildparameter 'buildSnapshotUUID' an.
