Для поддержки RTC SCM необходим доступ к комплекту инструментов компоновки на том узле, где выполняется компоновка. Можно настроить несколько версий комплектов инструментов компоновки (например, один для RTC 4.0.2, а второй - для RTC 3.0.1.5). Каждой установленной компоновке необходимо дать имя, чтобы облегчить выбор необходимого комплекта инструментов при настройке глобальных параметров, заданий и подчиненных узлов.
