La vía de acceso al kit de herramientas de compilación. Será un directorio de las herramientas de compilación de RTC instaladas. Ejemplo: C:\Archivos de programa (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

Si piensa cargar enlaces simbólicos, deberá realizar una configuración adicional. Para crear enlaces simbólicos es necesario tener acceso a las bibliotecas nativas. Puede obtener detalles adicionales sobre la configuración visitando Plugin de Team Concert
