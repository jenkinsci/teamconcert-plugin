Określ właściciela obrazu stanu. Obraz stanu może należeć do obszaru roboczego repozytorium lub strumienia. Określenie właściciela obrazu stanu umożliwia przeprowadzanie wyszukiwania o określonym zasięgu, dzięki czemu w wynikach wyszukiwania nie ma duplikatów.

Jeśli użytkownik nie ma szczegółowych informacji o właścicielu lub jeśli nazwa obrazu stanu jest na pewno unikalna w ramach repozytorium, wtedy można wybrać dla właściciela obrazu stanu wartość - brak -. Szczegółowe informacje o właścicielu obrazu stanu są ignorowane, jeśli zostanie podany identyfikator UUID obrazu stanu.

Jeśli dla właściciela obrazu stanu zostanie wybrana wartość - brak -, wyszukiwanie obrazów stanu nie będzie uwzględniało zasięgu i będzie dotyczyło całego repozytorium.

Jeśli zostanie podany obszar roboczy repozytorium lub strumień będący właścicielem, wyszukiwanie obrazów stanu będzie miało określony zasięg - tylko obrazy stanu powiązane z określonym obszarem roboczym repozytorium lub strumieniem.

Jeśli jako właściciel obrazu stanu zostanie wybrany obszar roboczy repozytorium, ale nie zostanie podana jego nazwa, wtedy wyszukiwanie obrazów stanu nie zostanie ograniczone zasięgiem i będzie dotyczyło całego repozytorium.

Jeśli jako właściciel obrazu stanu zostanie wybrany strumień, ale nie zostanie podany obszar projektu, obszar zespołu ani nazwa strumienia, wtedy wyszukiwanie obrazów stanu nie zostanie ograniczone zasięgiem i będzie dotyczyło całego repozytorium.

Jeśli jako właściciel obrazu stanu zostanie wybrany strumień i zostanie podana tylko nazwa obszaru projektu lub obszaru zespołu, wtedy wyszukiwanie obrazów stanu zostanie ograniczone do obrazów stanu powiązanych ze wszystkimi strumieniami należącymi do określonego obszaru projektu lub obszaru zespołu.

Jeśli jako właściciel obrazu stanu zostanie wybrany strumień i zostanie podana tylko nazwa strumienia, wtedy wyszukiwanie obrazów stanu zostanie ograniczone do strumienia o danej nazwie w repozytorium. Jeśli w repozytorium znajduje się wiele strumieni o tej samej nazwie, wyszukiwanie nie powiedzie się.

Jeśli jako właściciel obrazu stanu zostanie wybrany strumień i zostanie podana nazwa obszaru projektu lub obszaru zespołu, jak również nazwa strumienia, wtedy wyszukiwanie obrazów stanu zostanie ograniczone do obrazów stanu powiązanych ze strumieniem o określonej nazwie w określonym obszarze projektu lub obszarze zespołu. Jeśli w określonym obszarze projektu lub obszarze zespołu znajduje się wiele strumieni o tej samej nazwie, wyszukiwanie nie powiedzie się.
