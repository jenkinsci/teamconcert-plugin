O caminho para o Kit de Ferramentas de Compilação. Este será um diretório dentro das ferramentas de compilação do RTC instalado. Exemplo: C:\Program Files (x86)\IBM\TeamConcertBuild\buildsystem\buildtoolkit

O caminho é para o Kit de Ferramentas de Compilação no Principal. Se você tiver uma configuração de compilação distribuída (Escravos), também precisará especificar onde o Kit de Ferramentas de Compilação está no Escravo (assumindo que está em um caminho diferente). Após o salvamento, acesse a configuração do Escravo. Na seção Propriedades do Nó, inclua os locais da Ferramenta para cada um dos Kits de Ferramenta de Compilação a serem usados no Escravo. Consulte Configuração Principal/Escrava para obter detalhes.

Se você pretende carregar links simbólicos, é necessário fazer uma configuração adicional. O acesso às bibliotecas nativas é necessário para a criação de links simbólicos. Para obter mais detalhes sobre a configuração, visite Team Concert Plugin
