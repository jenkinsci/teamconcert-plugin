Outils de génération RTC à utiliser lors des générations. Les outils disponibles sont définis dans la configuration système (avec les autres outils tels que Ant et Java). Les outils de génération sont également requis sur le noeud maître pour l'interrogation et la validation de la configuration des travaux sauf si l'option "Eviter d'utiliser les outils de génération sur le noeud maître" est activée.
