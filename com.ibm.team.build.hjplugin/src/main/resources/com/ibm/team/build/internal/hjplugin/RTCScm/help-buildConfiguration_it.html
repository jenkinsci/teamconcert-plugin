Definizione build

La selezione di una definizione di buile consentirà l'integrazione con la build RTC. Ciò consentirà di richiedere le build da RTC o Hudson/Jenkins. Inoltre, i risultati della build verranno creati in entrambi e collegati tra loro.

Con l'opzione Jazz Source Control, è possibile specificare uno spazio di lavoro del repository insieme ad ulteriori opzioni di accettazione e caricamento.


Spazio di lavoro repository

Selezionando uno spazio di lavoro del repository, le serie di modifiche verranno accettate e richiamate dalle relative destinazioni flusso.

È possibile specificare il punto in cui verrà caricato lo spazio di lavoro del repository, se verranno create le cartelle del componente e se eliminare la sandbox prima del caricamento.


Istantanea

La selezione di questa opzione consente di caricare un'istantanea Jazz SCM.

È possibile specificare il punto in cui verrà caricato lo spazio di lavoro del repository, se verranno create le cartelle del componente e se eliminare la sandbox prima del caricamento.


Flusso

Selezionando uno stream, verrà creata un'istantanea ed il caricamento verrà eseguito dall'istantanea.

È possibile specificare il punto in cui verrà caricato lo spazio di lavoro del repository, se verranno create le cartelle del componente e se eliminare la sandbox prima del caricamento.
