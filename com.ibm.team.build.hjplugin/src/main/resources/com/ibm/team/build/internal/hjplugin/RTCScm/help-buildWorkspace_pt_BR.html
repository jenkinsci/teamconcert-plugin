O nome da área de trabalho do repositório de compilação dentro do servidor Rational Team Concert (RTC). Apenas uma área de trabalho com o nome deve ser definida. A área de trabalho do repositório de compilação deve colaborar com o(s) fluxo(s) a serem compilados.

Se você tiver requisitos específicos relacionados a como a área de trabalho do repositório de compilação deve ser carregada (exemplo: criar pastas de componentes, usar regras de carregamento, excluir ambientes de simulação antes do carregamento, etc.) crie uma definição de compilação que reflita os requisitos. Use a opção para compilar a partir de uma definição de compilação em vez da área de trabalho.
